# Tic Tac Toe Game in Java

A simple console-based Tic Tac Toe game built using Java.  
Two players can play turn by turn until one wins or the game ends in a draw.

## Features
- 2-player mode
- Console-based gameplay
- Checks win and draw conditions
- Easy to understand Java code

## How to Run
1. Install Java on your system.
2. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
3. Go to the project folder:
   ```bash
   cd your-repo-name
   ```
4. Compile the program:
   ```bash
   javac TicTacToe.java
   ```
5. Run the game:
   ```bash
   java TicTacToe
   ```

## How to Play
- Player 1 uses X
- Player 2 uses O
- Enter a number from 1 to 9 to choose a position on the board
- The first player to get 3 in a row wins
